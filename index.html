UCSC Puma Project - ML Intern

I worked with the UCSC Puma Project over the summer of 2016 to develop a machine learning algorithm for puma kill prediction. Throughout this process, I learned a lot about Machine Learning through online resources such as Coursera's Machine Learning Course and countless web articles. I ended up implementing the algorithm in R. Later, I summarized my project into a research paper and presented it to the community.

  • Date: Summer 2016
